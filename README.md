## 🚀 Installation (Global & Instant)

### 1. Clone or Download
```bash
git clone https://github.com/bobtech-IIT/grok-mode-skill.git
```

### 2. Global Install
Copy the `SKILL.md` to your skills directory:
```bash
mkdir -p ~/.claude/skills/grok-mode
cp SKILL.md ~/.claude/skills/grok-mode/SKILL.md
```

Or for project-specific:
```bash
mkdir -p .claude/skills/grok-mode
cp SKILL.md .claude/skills/grok-mode/SKILL.md
```

### 3. Activate
In Claude Code:
```bash
/reload-plugins
/grok-mode
```

It auto-loads on new sessions. Works identically inside Grok chats.

**Pro Tip:** Pair with Context Mode + Claude Mem plugins for full persistence power.

1. **Grill & Brainstorm** — Relentless interview + design approval
2. **Plan in Vertical Slices** — Exact files, tests, verification
3. **Optimize & Persist** — Sandbox + DB memory injection

5. **Review & Guard** — Code quality + git safety
6. **Handoff & Meta** — Continuity + new skill creation
